📌 使い方

以下のコマンドはターミナルで実行してください。コマンドの右上の「コピー」ボタンをクリックするとクリップボードにコピーできます。
URL を実際の YouTube URL に置き換えて実行してください。

動画ダウンロード（H.264）

📺 HD1080p でダウンロード
HD1080p（クッキーなし - 公開動画向け）
1080p（フルHD）の品質でダウンロード。H.264（mp4形式）で安定ダウンロード。
yt-dlp -f "bestvideo[vcodec^=h264][height<=1080][ext=mp4]+bestaudio[ext=m4a]/best[height<=1080][ext=mp4]" -o "%(title)s.%(ext)s" "URL"
HD1080p（Brave認証付き - 制限付き動画向け）
1080p（フルHD）の品質でダウンロード。Braveブラウザのクッキーで認証対応。H.264（mp4形式）で安定ダウンロード。
yt-dlp --cookies-from-browser brave -f "bestvideo[vcodec^=h264][height<=1080][ext=mp4]+bestaudio[ext=m4a]/best[height<=1080][ext=mp4]" -o "%(title)s.%(ext)s" "URL"
📱 SD720p でダウンロード
SD720p（クッキーなし - 公開動画向け）
720p（HD）の品質でダウンロード。H.264（mp4形式）でストレージ節約に最適。
yt-dlp -f "bestvideo[vcodec^=h264][height<=720][ext=mp4]+bestaudio[ext=m4a]/best[height<=720][ext=mp4]" -o "%(title)s.%(ext)s" "URL"
SD720p（Brave認証付き - 制限付き動画向け）
720p（HD）の品質でダウンロード。Braveブラウザのクッキーで認証対応。H.264（mp4形式）で確実ダウンロード。
yt-dlp --cookies-from-browser brave -f "bestvideo[vcodec^=h264][height<=720][ext=mp4]+bestaudio[ext=m4a]/best[height<=720][ext=mp4]" -o "%(title)s.%(ext)s" "URL"

動画ダウンロード（AV1優先）

🎬 最高品質でダウンロード
最高品質ダウンロード（クッキーなし - 公開動画向け）
公開されているYouTube動画を最高品質でダウンロードします。AV1優先、次点でH.264（mp4形式）で確実にダウンロード。
yt-dlp -f "bestvideo[vcodec^=av01][ext=mp4]+bestaudio[ext=m4a]/bestvideo[vcodec^=h264][ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]" -o "%(title)s.%(ext)s" "URL"
最高品質ダウンロード（Brave認証付き - 制限付き動画向け）
年齢制限やプライベート動画など認証が必要な動画をダウンロードします。Braveブラウザのクッキーを使用。AV1優先、次点でH.264（mp4形式）で確実にダウンロード。
yt-dlp --remote-components ejs:github --cookies-from-browser brave -f "bestvideo[vcodec^=av01][ext=mp4]+bestaudio[ext=m4a]/bestvideo[vcodec^=h264][ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]" -o "%(title)s.%(ext)s" "URL"
📺 HD1080p でダウンロード
HD1080p（クッキーなし - 公開動画向け）
1080p（フルHD）の品質でダウンロード。AV1優先、次点でH.264（mp4形式）で高品質と容量のバランスを実現。
yt-dlp -f "bestvideo[vcodec^=av01][height<=1080][ext=mp4]+bestaudio[ext=m4a]/bestvideo[vcodec^=h264][height<=1080][ext=mp4]+bestaudio[ext=m4a]/best[height<=1080][ext=mp4]" -o "%(title)s.%(ext)s" "URL"
HD1080p（Brave認証付き - 制限付き動画向け）
1080p（フルHD）の品質でダウンロード。Braveブラウザのクッキーで認証対応。AV1優先、次点でH.264（mp4形式）で安定ダウンロード。
yt-dlp --remote-components ejs:github --cookies-from-browser brave -f "bestvideo[vcodec^=av01][height<=1080][ext=mp4]+bestaudio[ext=m4a]/bestvideo[vcodec^=h264][height<=1080][ext=mp4]+bestaudio[ext=m4a]/best[height<=1080][ext=mp4]" -o "%(title)s.%(ext)s" "URL"
📱 SD720p でダウンロード
SD720p（クッキーなし - 公開動画向け）
720p（HD）の品質でダウンロード。AV1優先、次点でH.264（mp4形式）でストレージ節約に最適。
yt-dlp -f "bestvideo[vcodec^=av01][height<=720][ext=mp4]+bestaudio[ext=m4a]/bestvideo[vcodec^=h264][height<=720][ext=mp4]+bestaudio[ext=m4a]/best[height<=720][ext=mp4]" -o "%(title)s.%(ext)s" "URL"
SD720p（Brave認証付き - 制限付き動画向け）
720p（HD）の品質でダウンロード。Braveブラウザのクッキーで認証対応。AV1優先、次点でH.264（mp4形式）で確実ダウンロード。
yt-dlp --remote-components ejs:github --cookies-from-browser brave -f "bestvideo[vcodec^=av01][height<=720][ext=mp4]+bestaudio[ext=m4a]/bestvideo[vcodec^=h264][height<=720][ext=mp4]+bestaudio[ext=m4a]/best[height<=720][ext=mp4]" -o "%(title)s.%(ext)s" "URL"

音声抽出

MP3形式で音声のみ抽出
動画から音声のみを抽出して MP3 形式で保存します。音楽の動画に最適です。
yt-dlp -f bestaudio --extract-audio --audio-format mp3 "URL"
M4A形式で音声抽出
動画から音声のみを抽出して M4A 形式（Apple 形式）で保存します。
yt-dlp -f bestaudio --extract-audio --audio-format m4a "URL"

詳細・応用機能

💾 ファイル名指定
カスタムファイル名で保存
デフォルトの YouTube タイトルではなく、カスタムファイル名で保存します。AV1優先、次点でH.264（mp4形式）で確実ダウンロード。
yt-dlp -f "bestvideo[vcodec^=av01][ext=mp4]+bestaudio[ext=m4a]/bestvideo[vcodec^=h264][ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]" -o "%(title)s.%(ext)s" "URL"
🎞️ 映像と音声の合成
映像と音声を合成
映像と音声がバラバラにダウンロードされた場合、ffmpeg で映像と音声を合成して一つのファイルにします。
cd ~/Desktop/move && ffmpeg -i "aaa.mp4" -i "aaa.m4a" -c copy "aaa_merged.mp4"
📋 プレイリスト
プレイリスト全体ダウンロード
YouTube プレイリストの全ての動画をダウンロードします。個別の URL ではなくプレイリストの URL を使用。AV1優先、次点でH.264（mp4形式）で確実ダウンロード。
yt-dlp -f "bestvideo[vcodec^=av01][ext=mp4]+bestaudio[ext=m4a]/bestvideo[vcodec^=h264][ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]" "Playlist_URL"
ℹ️ 情報表示
特定のフォーマット一覧を表示
その動画で利用可能なすべてのフォーマット（解像度、ファイル形式など）を表示します。ダウンロード前に品質を確認する時に使用します。
yt-dlp -F "URL"
